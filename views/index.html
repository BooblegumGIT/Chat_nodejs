Чат

Чтобы посмотреть работу чата на локальной машине, просто запустите чат в сосдених вкладках Вашего браузера.

Инструменты

Демонстрационная программа "Чат".
Серверная часть написана на nodejs с использованием модулей:
  • Express
  • Socket.io
  • Шаблонизатор swig
  • Morgan для удобных логов
Клиентская часть на javascript ( JQuery)

Структура проекта сгенерирована IDE WebStorm .

P.S.

Просьба не судить строго, с nodeJs знаком совсем недавно. "Дизайн" тоже делался в очень сжатые сроки (30 минут).
